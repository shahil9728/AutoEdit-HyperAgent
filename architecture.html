autoedit · system architecture

One pipeline, six agents,
one shared edit list.

Raw clips and a few user choices go in the top. Each agent makes one decision and writes it to a shared Edit Decision List (EDL) — the single source of truth — before the render layer turns that list into finished videos. Nothing is a black box: you can read or override the EDL at any stage.

The flow
↕ EDL bus — shared state
⬆️

Input

Multiple raw clips (a few seconds to several hours) + the user's choices: output format(s), captions, voice-over, music, auto-reframe.

🔍

01 Analysis Agent

Probes every file, transcribes speech to word-level timestamps, and now analyses the picture — splits footage into shots and scores each for sharpness, exposure, saturation and motion. Builds the structured understanding everything downstream reads.

metadata · real (ffprobe)visual quality · real (ffmpeg)transcription · Whisper-pluggableaesthetic model · planned
⭐

02 Clip Selection Agent

Ranks each shot by a blend of visual quality and speech engagement, weighted per format — cinematic/travel trust the picture, a vlog trusts the talking — then drops the weak / dark / blurry / low-value shots against a time budget.

blended visual + speech · realLLM ranking · pluggable
🎬

03 Storytelling Agent

Orders the kept clips into a sequence with pacing — leads with the strongest hook, then keeps the narrative coherent. Adapts to the chosen style.

rule-based · realstyle-aware · pluggable
✂️

04 Editing & Reframe Agent

Cuts each selected range, reframes to the target aspect (9:16 / 1:1 / 16:9), concatenates, and applies transitions / speed.

cut + reframe · real (ffmpeg)face-tracking reframe · planned
🔊

05 Audio, Captions & Voice-Over Agent

Generates word-pop captions, adds background music, and (new) can synthesize a voice-over track, ducking music underneath it. Syncs cuts to the beat.

captions · real (libass)voice-over (TTS) · newmusic + beat-sync · planned
📐

06 Platform Optimization Agent

Renders a tailored version per platform — aspect ratio, length, caption style and encode settings for Shorts, Reels, TikTok, feed, or cinematic.

presets + export · real
⬇️

Output

Finished, publish-ready videos — multiple platform versions from one run, plus the EDL JSON for review or manual tweaks.

Knobs we can turn

Selection brain swap

Heuristic → an LLM that reasons about hooks, payoff and emotion. One function (select_highlights).

Voice-over new

Add a TTS narration stage + auto-duck music. Script can be generated from the transcript or user-supplied.

Output presets

Add / edit formats (cinematic, travel montage, ad, product). Each is just an entry in presets.py.

Caption style

Word-pop, karaoke, or clean lines; fonts, colors, position. Driven by the target preset.

Reframe intelligence upgrade

Center-crop → face / saliency tracking so the subject stays framed (sets focus_x per clip).

Music & beat-sync

Licensed library + snap cuts to the beat. Licensing is the real constraint for monetized output.

Pacing & speed

Speed ramps, tighter cuts, silence trimming — tunable per style in the Storytelling agent.

Brand kit

Intro/outro, logo bug, lower-thirds, color LUT — a template layer on the Editing agent.
